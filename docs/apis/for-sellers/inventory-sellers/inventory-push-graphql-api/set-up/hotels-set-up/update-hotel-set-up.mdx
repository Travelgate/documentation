# Update Hotel

The `updateHotelSetUp` allows you to update a configuration from a hotel already added in a specific client - seller relation in Inventory. The returned fields include:

* `id`
* `hotelCode`
* `contextCode`
* `active`
* `sendNotification`
* `emailNotification`
* `emailNotificationCc`
* `emailNotificationSupport`

## Mutation Inputs

When creating your `updateHotelSetUp` mutation, you have 1 input to fill based on your specific needs:

1. Input

### 1. Input   

**Mandatory input**
* `clientCode`
* `supplierCode`
* `hotelCode`

**Optional input**
* `contextCode`
* `active`
* `sendNotification`
* `emailNotification`
* `emailNotificationCc`
* `emailNotificationSupport`
