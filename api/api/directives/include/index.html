include

Directs the executor to include this field or fragment only when the if argument is true.

directive @include(if: Boolean!) on FIELD | FRAGMENT_SPREAD | INLINE_FRAGMENT

Arguments​

include.if ● Boolean! non-null scalar​

Included when true.
