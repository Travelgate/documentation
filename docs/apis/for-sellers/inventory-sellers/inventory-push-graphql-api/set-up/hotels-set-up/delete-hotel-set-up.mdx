# Delete Hotel

The `deleteHotelSetUp` is designed to remove a specific hotel from a client - seller relation in Inventory. The returned fields include:

* `success`
* `adviseMessages`

## Mutation Inputs

When creating your `deleteHotelSetUp` mutation, you have 1 input to fill based on your specific needs:

1. Input

### 1. Input   

**Mandatory input**
* `clientCode`
* `supplierCode`
* `hotelCode`

**Optional input**
* `contextCode`
