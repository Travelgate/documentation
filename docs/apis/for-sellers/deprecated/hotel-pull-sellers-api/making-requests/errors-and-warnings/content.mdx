# Content

Below you will find a breakdown of the different errors that exist in the API to be returned to customers in the different content call responses. 
Clients are familiar with these types of errors and if they are well categorised they can help to highlight potential problems in property loading.

### Error List

| Code   | Description                                                                                                                                                          |
| -----  |----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 101    | Internal Error. System Exception (Exception not controlled or not classified as general exception).           |
| 201    | Error retrieving data.                                                                                        |
| 207    | Request not supported. The required parameter 'HotelXAccessCode' is missing from the configuration parameters.|
| 302    | Hotel Not Found in DescriptiveInfo.                                                                           |
