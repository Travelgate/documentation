# Get Clients

The `allClients` query returns all Clients (service accounts) available in your organization.

A `client` identifies the application or integration component that makes requests to the platform.

Use this endpoint when you need to:

* Audit available client identities
* Verify which clients are active
* Inspect the group assigned to each client

## Query Overview

Path: `admin/get-clients/`

GraphQL root:

```graphql
admin {
  allClients {
    edges {
      node {
        clientData {
          ...
        }
      }
    }
  }
}
```

The response is a connection, so you can read:

* `edges`: list of clients
* `totalCount`: total number of clients in your organization

## Response Considerations

* `code` follows the client identifier format used by the platform.
* `group` helps understand where the client belongs in your organization hierarchy.
* Depending on your role permissions, some sensitive fields may not be returned.
