# Quickstart

## API Endpoint

All Social API operations use the following endpoint:

```
https://api.travelgate.com
```

## Overview

The Social API lets you retrieve profile and relationship data for organizations in the Travelgate ecosystem.

### Common Use Cases

- **List Profiles**: Retrieve profile information, optionally filtered by owner code.
- **Inspect Commercial Status**: Review buyer and seller status, lifecycle, and integration fields.
- **Review Contacts and Classifications**: Access profile contacts, categories, markets, and classifications.

## Authentication

All Social API calls require JWT (JSON Web Token) authentication.
