## **Overview**

The `Categories` operation allows clients to retrieve categories from suppliers. The returned fields include: code and name.

## **Request**

The Request is structured to accommodate access information for the content retrieval.

### **Overview**

The request that receives the Connector is a Task with essential information that allows it to execute the operation properly:
- **Access**: Represents access credentials and related information required for a connection.
- **Operation**: Operation to be executed.

### **Sample**

```json
{
    "Task": {
        "Access": {
            "Code": 11111,
            "User": "tgx",
            "Password": "password",
            "Urls": {
                "Generic": "https://api.hotelsupplier.com/generic"
            },
            "Parameters": {
                "maxResults": "50",
                "cacheEnabled": "true",
                "requestTimeout": "5000",
                "preferredSupplier": "SupplierX"
            }
        },
        "Operation": "Boards",
        "Languages": [ "en" ],
        "Platform": "HOTELTEST",
        "Supplier": "HOTELTEST",
        "SupplierGroup": "HOTELTEST"
    }
}
```

The object [ContentRq](/docs/apis/for-sellers/connectors-pull-developers-api/api-reference/contentrq) represents the structure of the content request as expected by the supplier.

## **Response**

The response contains the mapped categories information.

### **Overview**

The response provides a straightforward structure for mapping categories across multiple languages.

### **Sample**

```json
{
	"Code": "2",
	"Names": {
		"en": "2 Stars"
	}
}
```

### **Details**

The [AccumulativeCategory](/docs/apis/for-sellers/connectors-pull-developers-api/api-reference/accumulativecategory) object represents the standardized structure of the mapped category element.

<!-- TABLE BEGIN Content.Common.Domain.AccumulativeContracts.Category -->

| **Code** | String | Code of the category. |
| **Names** | Dictionary&lt;[Iso2LanguageEnum](/docs/apis/for-sellers/connectors-pull-developers-api/api-reference/iso2languageenum), String&gt;  | Dictionary of category names by ISO 2 language. |
