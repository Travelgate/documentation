# 101 Error - Internal error

## What Does a 101 Error Mean?
A **101 error** indicates an internal issue in Travelgate's connector service, usually caused by an unexpected exception.

#### Example:

```json
 "errors": [
            {
              "code": "22101",
              "description": "Error Description",
              "type": "101"
            }
          ]
```

## How to Resolve a 101 Error

#### 1. Contact Support
Open a case through the [Travelgate Support Team](https://app.travelgate.com/support).

#### 2. Share Technical Context
Include the logs and all relevant details about the scenario where the error was detected (operation, request context, and timing). Our team will review the case and work on the resolution as soon as possible.
