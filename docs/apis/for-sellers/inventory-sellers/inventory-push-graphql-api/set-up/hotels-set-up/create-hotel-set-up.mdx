# Create Hotel

The `createHotelSetUp` allows you to add a new hotel, with its configuration, to a specific client - seller relation in Inventory. The returned fields include:

* `id`
* `hotelCode`
* `contextCode`
* `active`
* `sendNotification`
* `emailNotification`
* `emailNotificationCc`
* `emailNotificationSupport`

## Mutation Inputs

When creating your `createHotelSetUp` mutation, you have 1 input to fill based on your specific needs:

1. Input

### 1. Input   

**Mandatory input**
* `clientCode`
* `supplierCode`
* `hotelCode`
* `active`
* `sendNotification`

**Optional input**
* `contextCode`
* `emailNotification`
* `emailNotificationCc`
* `emailNotificationSupport`
