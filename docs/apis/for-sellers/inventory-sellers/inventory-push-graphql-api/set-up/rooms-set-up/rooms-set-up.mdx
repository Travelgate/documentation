# Rooms

The `roomsSetUp` query returns the rooms added to a hotel rate in Inventory. The returned fields include:

* `code`
* `master`
	* `id`
	* `code`
	* `name`
* `active`
* `externalCode`
* `standard`
* `uses`
	* `numberOfGuests`
	* `minAge`
	* `maxAge`
	* `paxType`

## Query Inputs

When creating your `roomsSetUp` query, you have 1 input to fill based on your specific needs:

1. Criteria

### 1. Criteria   

**Mandatory input**
* `clientCode`
* `supplierCode`
* `hotelCode`
* `rateCode`

**Optional input**
* `contextCode`
* `roomCodes`
