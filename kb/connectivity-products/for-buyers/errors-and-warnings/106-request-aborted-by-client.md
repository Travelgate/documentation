# 106 Error - Request aborted by client

## What Does a 106 Error Mean?
The **106 - Request aborted by client** error is a communication error that occurs when the Buyer (client) closes the connection for a transaction through Travelgate before a response is received from our system.

:::info
If you observe a significant spike in 106 errors, please contact our [Support Team](/kb/platform/support-portal/how-to-submit-case) for assistance.
:::
