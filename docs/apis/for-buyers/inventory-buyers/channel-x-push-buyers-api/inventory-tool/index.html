Inventory Tool

Since Channel-X solution is integrated with Inventory Tool, first of all you must use our Inventory extranet in order to configure the setup for the hotels, rate plans and rooms contracted with the hoteliers, so that your Channel Managers can map them as well. Once mapped, the Channel Manager will load and update availability and prices for the hotels, rate plans and rooms configured.

tip

To learn how to setup hotels, rate plans, and rooms in the Inventory Tool, please refer to the Inventory SetUp section.

¬†

After completing this preliminary step, you will be able to perform a Full Copy. From that point onward, you will receive smaller requests, known as Deltas, to stay informed about updates.
