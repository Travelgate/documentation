# Quickstart

## Introduction

[Inventory](https://inventory.xmltravelgate.com/Account/Login?ReturnUrl=%2f) enables you to connect with major Channel Managers, allowing you to focus on what truly matters most â€“ contracting! In our extranet, we provide you with tools to set up hotels, rates, and rooms contracted with hoteliers, create user accesses for your providers, and check your bookings, among other options.

:::tip INFO


Buyers connecting with Push Sellers, utilizing the Inventory tool is essential for setting up hotels, rate plans, and rooms, as Sellers handle the push availability and prices.

Partners without an API, opting for Inventory + Distribution combination, need to manually set up hotels, rate plans, and rooms, as well as manually load availability and prices.

:::

## Let's Get Started: How to Set Up the Inventory APP!

In this quickstart guide, we will show you the main steps to set up your Inventory account, allowing you to start buying as quickly as possible. Inventory empowers Buyers to enhance bookings by capitalizing on Channel Managers. It also facilitates Sellers without an API in uploading their own products, in combination with our Distribution app, for seamless sales to Buyers.

### 1. Connect with Sellers (or yourself).

To purchase from Push Sellers or upload your own product to Travelgate is essential to establish connections with them. Explore the [step-by-step guide in the following article](../../../kb/connections/my-connections/guick-guide-to-auto-activations), outlining how to easily connect with different Sellers on the [Travelgate website](https://app.travelgatex.com/connections/myconnections).

![MyConnections_Distribution](https://storage.travelgate.com/docs/MyConnections_Distribution.png)


### 2. Carry out the setup for hotels, rate plans, and rooms.

Use [Inventory extranet](https://inventory.xmltravelgate.com/Account/Login?ReturnUrl=%2f) to [set up hotels, rate plans, and rooms](../inventory/extranet/set-up/setup) contracted with hoteliers.

![SetUp Inventory](https://storage.travelgate.com/docs/setup-inventory.png)


:::info

If you'll **only be using Inventory tool to connect with Push Sellers**, that's all for you! The Seller will push availability and prices for you and you will be able to start Buying! 

:::

### 3. Manually upload both availability and prices.

Afterward, you will need to [manually load and update availability and prices](../inventory/extranet/availability-and-rates/manual-load/overview) for the configured hotels, rate plans, and rooms.

![Manual Load](https://storage.travelgate.com/docs/ManualLoad.png)
