# Overview

Master CSV files provide the essential static Inventory information.
To effectively manage this data, please familiarise yourself with the `roomsMaster` and `hotelsMaster` files.

:::important Always upload Masters first
The setup files (`hotels`, `rateplans`, `rooms`, `policies`, `taxes`, `offers`) reference entries declared in the master files. Always upload `hotelsMaster` and `roomsMaster` **before** the corresponding setup files. See the [Set Up overview](../set-up/overview) for the full recommended upload order.
:::
