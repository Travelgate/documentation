# Chains

The Chains Query allows you to retrieve hotel chain master data in two ways:
- The **FastX Chain List**, which returns standardized chain values from Travelgate.
- The **Supplier Chain List**, which returns native chain values configured by each connected Seller.

In the Hotels list response, you receive `chainCode` for each property. The list of possible chain values depends on the access used in the query (FastX access or Supplier access).

The returned fields include:

* `chainCode`
* `text`
* `language`

## Query Overview

To retrieve chain values, use the Chains Query with the `criteria` input.

When creating your chains query, you have one input to fill based on your needs:

1. Criteria

### 1. Criteria

**Mandatory criteria**
* `access`

**Optional criteria**
* `chainCodes` (Search by chain code)

Some hotels may not include a `chainCode` in the hotel list response.

:::
