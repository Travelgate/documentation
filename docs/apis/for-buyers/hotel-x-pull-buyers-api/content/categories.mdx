# Categories

The Categories Query allows you to retrieve category information in two ways:
- The **FastX Category List**, which returns standardized categories from Travelgate.
- The **Supplier Category List**, which returns the native category values configured by each connected Seller.

Categories represent the classification types that can be assigned to a hotel (for example, star ratings and similar category systems).

In the Hotels list response, you receive the `category` field for each property. The possible category values that may appear there are the ones listed in this Categories call, either from the FastX master or from the Supplier master, depending on the access used.

The returned fields include:

* `categoryCode`
* `text`
* `language`


## Query Overview

When creating your categories query, you have 1 input to fill based on your specific needs:

1. Criteria

### 1. Criteria    

This query offers versatility, with certain fields marked as mandatory (`access`) and others as optional (`categoryCodes` and `group`). This flexibility empowers you to create a personalized Categories Query, tailoring the requested fields to your specific needs. Consequently, the response will only include information that is highly relevant to you, streamlining the retrieval process and ensuring the data you receive meets your exact requirements.

**Mandatory criteria**
* `access`

**Optional criteria**
* `categoryCodes` (Search by category type)
* `group`

## Use Cases

### FastX master category list

To retrieve the **FastX Category Master**, make a `category list` call using your **FastX access**, which you can find in the *MyConnections* section under the Seller **"Travelgate"**.

<GraphqlSample query={categoriesQuery} variables={categoriesFastXVariables} />


### Supplier category list

To retrieve one Supplier's native category master, include that Supplier's access in the query (for example, access `2` for the Travelgate Test supplier).

<GraphqlSample query={categoriesQuery} variables={categoriesVariables} />

:::note

The amount of information and content returned might vary between Sellers.

Hotel-X supports all languages, but receiving category descriptions in one language or another depends entirely on the languages supported by the Seller.

:::


### Category list for specific category codes

Use the Categories List Query to filter specific category types. For example, to focus on 4-star inventory from a Supplier, include `categoryCodes` such as "4" and "5" in your criteria (these are supplier-native category codes in this example). This helps you retrieve more targeted and relevant results.
