# Hotel Categories

The `hotelCategories` query returns a list of the categories that a hotel can have assigned in Inventory. The returned fields include:

* `id`
* `name`


## Query Inputs

When creating your `hotelCategories` query, you have 1 input to fill based on your specific needs:

1. Criteria

### 1. Criteria   

This query offers versatility, with an optional field `categoryIds` that can be filled with the categories to retrieve.

**Optional input**
* `categoryIds`
