# Get Suppliers

The `allSuppliers` query returns all Suppliers available in your organization.

A `supplier` represents the Seller connection metadata used across your Hotel-X integrations.

Use this endpoint when you need to:

* Identify which suppliers are currently active
* Map supplier `code` with its `context`
* Validate provider-level information for troubleshooting

## Query Overview

Path: `admin/get-suppliers/`

GraphQL root:

```graphql
admin {
  allSuppliers {
    edges {
      node {
        supplierData {
          ...
        }
      }
    }
  }
}
```

The response is a connection, so you can read:

* `edges`: list of suppliers
* `totalCount`: total number of suppliers in your organization

## Response Considerations

* `isActive` can be affected by provider status.
* `context` is useful to configure plugins and static-content mappings.
* `serviceApi` and `api` help identify the technical integration type per supplier.
