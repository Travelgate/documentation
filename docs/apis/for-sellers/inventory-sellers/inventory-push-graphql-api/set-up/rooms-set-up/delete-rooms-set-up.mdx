# Delete Rooms

The `deleteRoomsSetUp` is designed to remove rooms from hotel rates in Inventory. The returned fields include:

* `success`
* `adviseMessages`

## Mutation Inputs

When creating your `deleteRoomsSetUp` mutation, you have 1 input to fill based on your specific needs:

1. Input

### 1. Input   

**Mandatory input**
* `clientCode`
* `supplierCode`
* `hotelCode`
* `rateCode`
* `roomCodes`

**Optional input**
* `contextCode`
